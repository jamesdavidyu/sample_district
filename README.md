# sample_district
An example of a Planorban project consulting a school district client. 
# intro
The purpose of this visualization is to report a bivariate analysis on how the English Language Arts (ELA) or Math proficiency scores of each and every school in an anonymous school district depends on the student and/or staff positivity scores from the same schools in the district. This visualization is for demonstration only so all data that are not publicly accessible has been fabricated. Even some data that are publicly accessible has been anonymized to protect the identity of the visualization’s subject.
